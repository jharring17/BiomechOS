# BiomechOS

## How to use our project

- Download the bash script **bash_script.sh**, open your command prompt (CMD) and run the script so all of the library imports that we use work for your computer.

- Click on and download the **plotBasic.py** file and make sure the data you want to use is stored in a similar directory layout to the one in this repo.

- Navagate yourself in the CMD to the folder that the python file is stored in, run **python plotBasic.py** followed by the folder where your data is stored.

- For example, **python plotBasic.py /Data/Nairobi** would run the data and display the data under that folder.
